## 프로젝트 개요

게임 퍼블리싱 회사의 번역 비용 절감을 위해 개발된 사내 전용 AI 번역 웹 툴.
Claude API를 활용하여 게임 콘텐츠 특화 번역, 용어 일관성 관리, 품질 검수까지 원스톱으로 처리한다.

## 주요 기능

### 1. 프로젝트 관리
게임 프로젝트별로 번역 작업을 독립적으로 관리. 각 프로젝트에 소스 언어, 대상 언어, 콘텐츠 유형을 설정한다.

### 2. 파일 업로드 및 컬럼 매핑
Excel (.xlsx) / CSV 파일을 업로드하면 컬럼을 자동 감지하고, 사용자가 소스 텍스트 컬럼을 지정하는 매핑 UI를 제공한다.

### 3. AI 번역 엔진
Claude API를 사용하여 콘텐츠 유형별 특화 프롬프트로 번역 수행:
- **UI 텍스트** — 버튼, 메뉴 등 간결한 인터페이스 텍스트
- **스토리/내러티브** — 게임 스토리, 대사, 세계관 텍스트
- **아이템명** — 아이템, 스킬, 캐릭터 고유명사
- **시스템 메시지** — 알림, 오류, 안내 메시지

용어집 자동 주입 및 번역 메모리 참조로 일관성 보장.

### 4. 번역 메모리 (TM)
Levenshtein 알고리즘 기반 퍼지 매칭으로 이전 번역 재사용. 유사도 임계값 조정 가능. 승인된 번역은 자동으로 TM에 저장된다.

### 5. 용어집 관리
게임별 전문 용어 등록 및 관리. Excel 가져오기/내보내기 지원. 번역 시 자동으로 용어집을 참조하여 고유 용어가 일관되게 번역된다.

### 6. 검토 시스템
원문/번역문 나란히 비교 뷰. 인라인 편집, 승인/거부 처리. 승인된 항목은 자동으로 번역 메모리에 추가된다.

### 7. QA 검사 엔진
6개 자동 검사 모듈로 번역 품질을 검증:

| 모듈 | 설명 |
|------|------|
| 플레이스홀더 검사 | `{변수}` 누락 또는 불일치 감지 |
| 길이 검사 | 원문 대비 번역 길이 비율 이상 감지 |
| 용어집 검사 | 지정 용어 미사용 경고 |
| 누락 번역 검사 | 번역되지 않은 항목 감지 |
| 서식 검사 | HTML 태그 및 마크업 일치 검증 |
| 숫자 검사 | 숫자 누락 또는 변경 감지 |

## 페이지 구성

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 프로젝트 목록 | `/` | 전체 프로젝트 카드 그리드 |
| 프로젝트 대시보드 | `/projects/[id]` | 번역 진행률, 통계 |
| 번역 위자드 | `/projects/[id]/translate` | 4단계: 업로드 → 매핑 → 번역 → 결과 |
| 검토 페이지 | `/projects/[id]/review` | 나란히 비교, 승인/거부 |
| 용어집 | `/projects/[id]/glossary` | 용어 CRUD, Excel 가져오기/내보내기 |
| 번역 메모리 | `/projects/[id]/translation-memory` | TM 조회 및 관리 |
| QA 검사 | `/projects/[id]/qa` | 자동 품질 검사 실행 |
| 설정 | `/settings` | API 키, 기본 언어 설정 |

## 데이터베이스 구조

SQLite 기반, 8개 테이블:

| 테이블 | 설명 |
|--------|------|
| `Project` | 게임 프로젝트 정보 |
| `TranslationItem` | 번역 항목 (원문 + 번역문) |
| `GlossaryTerm` | 프로젝트별 용어집 항목 |
| `TranslationMemory` | 승인된 번역 메모리 |
| `QAResult` | QA 검사 결과 |
| `FileUpload` | 업로드된 파일 메타데이터 |
| `Language` | 지원 언어 목록 |
| `Settings` | 앱 전역 설정 |

## Claude API 활용 방식

- **모델:** `claude-sonnet-4-5`
- **번역 방식:** 콘텐츠 유형별 특화 시스템 프롬프트
- **용어집 주입:** 시스템 프롬프트에 프로젝트 용어집 포함
- **TM 참조:** 유사 번역 예시를 few-shot으로 제공
- **배치 처리:** 여러 항목을 JSON 배열로 묶어 단일 API 호출로 처리
- **언어 지원:** 한국어 ↔ 영어, 일본어, 중국어(간체/번체), 태국어, 베트남어 등 글로벌 다국어
