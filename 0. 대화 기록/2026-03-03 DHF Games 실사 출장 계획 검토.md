## 대화 내용

### 출장 개요
- 방문사: DHF Games (IFC 푸저우)
- 일정: 3/18(수)~3/20(금) 2박 3일
- 인원: 3인 (개발 김창영 / 사업 황정원 / 지원 주원석) + 중국 지사 2인
- 목적: 1차 계약금 지급 전 리스크 검증

### 주요 피드백

**1. 일정 구조 우려**
- 실질 업무일이 Day 2 하루뿐 → 기술 실사 + 사업 + 토크노믹스를 하루에 몰아넣기 어려움
- Day 3 실제 가용 시간 약 2시간 (11시 마무리 → 공항 이동)
- 권장: 1박 연장(3박 4일) 또는 Day 1 야간 내부 정렬 미팅

**2. Go/No-Go 판단 기준 부재**
- 기능 구현율, Core Loop 완성도, 동시접속 테스트 기준 등 사전 정의 필요
- 실사 결과 보고서 템플릿을 출발 전에 준비하여 현장 즉시 기입

**3. WEB3 논의 깊이 부족**
- 토크노믹스를 별도 세션으로 분리 권장
- 확인 필요: 토큰 이코노미, 온체인/오프체인 구분, NFT/FT 범위, 지갑 UX, 규제 리스크
- Nexus vs DHF WEB3 역할 분담 라인 확정 필요

**4. 인원 구성**
- 기술 검증 1인 전담은 범위 과다 → 중국 지사 기술인력 서포트 투입 권장

**5. 추가 권장**
- 출장 전 DHF에 사전 자료 요청 (빌드 영상, GDD, 기술 아키텍처, 팀 조직도, 재무)
- Day 2 저녁을 비공식 리스크 파악 기회로 활용
