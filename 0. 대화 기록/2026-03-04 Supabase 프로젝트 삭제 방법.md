## 대화 내용

### 삭제 절차
1. app.supabase.com 대시보드 접속
2. Project Settings → General 탭
3. "Delete project" 클릭 → 프로젝트명 입력으로 확인
4. **비가역적 삭제** — DB, Storage, Auth 사용자 전부 삭제됨

### 주의사항
- 삭제 전 필요한 데이터 반드시 백업
- 연관된 외부 서비스 연동 해제 확인
