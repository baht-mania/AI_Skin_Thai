## 대화 내용

### 프로젝트 개요
- 5,000만원 투자 (YH Entertainment 아이돌 그룹 일본 활동 독점권, VENEW 경유)
- 수익 구조: 총매출의 12%가 투자자 풀 → 투자 비율대로 분배
- 정산 주기: 투어는 종료 후 월별, 기타(앨범 등)는 분기별
- 이행 의무: 3년간 투어 6회 + 앨범 6장

### PRD 반복 개선 과정
- v1.0 → v2.2까지 외부 피드백(ChatGPT 포함) 반영하며 진화
- 환율 정책 명확화: 고정 환율 없음, 투자비율은 지급시점 확정, 실제 분배는 엔→원 정산시점 환율

### v2.2 최종 주요 기능
- Decimal.js 기반 금융 정밀도
- 자동 정산 계산 + 수동 조정 추적
- 뉴스 인텔리전스: 매일 10시 스크래핑 + 규칙 기반/AI 분류
- 멤버 관리: 코어 6명 + 신규 1~2명
- 원천징수 필드, 정산 커뮤니케이션 로그
- 지연 민감도 분석, 전체 데이터 백업/복원
- 단일 사용자 전용 (피드백 메커니즘 불필요)
