## 대화 내용

### 사용자 요청
> "나는 노무사 시험을 준비하고 싶어. 노무사 시험에 대해서 리서치하고 정보를 토대로 프로덕트를 만들고 싶어. 최대한 PRD까지 완벽하게 만들어서 클로드 코드로 만들려고 해."

### 프로덕트 방향 결정
- 유형: AI 기반 노무사 학습앱 (문제풀이 + 해설)
- 타겟: 본인 직접 사용 (혼자 공부)
- 스택: Next.js + Supabase (풀스택 웹)

### PRD v1.0 → v1.1 주요 내용
- 시험 분석: 2차 합격률 8.96%, 서술형 논술 난이도 극심
- 핵심 차별점: "1.5차 병행 학습" — 1차 기출 풀이 중 2차 쟁점 연계
- MVP 기능: 기출 풀이, AI 해설(Claude API), 키워드 브릿지, 백지 복습, 약점 레이더, 복습 스케줄
- DB 스키마: questions, user_answers, second_exam_bridge, ai_explanations
- 비용: 월 $5~15 (Supabase Free + Claude API + Vercel Hobby)

### GPT 피드백 반영 사항
- ✅ 반영: 1.5차 병행 학습 포지셔닝, 키워드 브릿지, 백지 복습, 찍기 감지, PWA
- ⚠️ 조정: PWA 오프라인은 AI 해설 제외, 두문자 암기장은 v2.0으로
