# 2026-03-22 Consilium 운영 교훈

## ⛔ 절대 하지 말 것

### 1. 시뮬레이션 진행 중 서버 재시작 금지
- MiroFish는 **인메모리 저장 구조** → 서버 재시작 = 모든 시뮬레이션 데이터 소실
- 환경변수 변경, Restart service, Redeploy 모두 데이터 리셋 유발
- **시뮬레이션 완료 후에만** 서버 작업할 것

### 2. 무료 플랜 한계 사전 확인
- Zep FREE: episode 총량 한도 있음 (403 에러)
- Render FREE: 15분 sleep → 인메모리 데이터 소실
- OpenRouter: 잔고 부족하면 중간에 끊김

## 📋 오늘 실패 타임라인

| 시간 | 실패 | 원인 |
|------|------|------|
| 1차 | 시뮬레이션 404 | Render 무료 sleep → 인메모리 소실 |
| 2차 | Zep 403 | FREE 플랜 episode 한도 초과 |
| 3차 | 시뮬레이션 idle | Zep API 키 교체 후 서버 재시작 → 인메모리 소실 |

## ✅ 해결된 것
- Render Starter 업그레이드 ($7/월) → sleep 없음
- Zep 새 계정 → episode 한도 리셋
- LLM 429 retry 로직 추가 (30/60/90/120s 점진적 대기)
- Episode polling 실패 보호 (15회 연속 실패 시 중단)
