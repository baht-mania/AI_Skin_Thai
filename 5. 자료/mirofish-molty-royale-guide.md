# MiroFish × Molty Royale 중국 진출 시뮬레이션 실행 가이드

> **목적**: Molty Royale(AI 에이전트 배틀로얄 플랫폼)의 중국 시장 진출 시 중국 유저 반응/여론을 MiroFish로 시뮬레이션
> **작성**: Opus (설계/지시) → **실행 담당: Sonnet**
> **작성일**: 2026-03-19

---

## 1. MiroFish 로컬 환경 세팅

### 1.1 사전 요구사항

| 도구 | 버전 | 설치 확인 |
|------|------|----------|
| Node.js | 18+ | `node -v` |
| Python | 3.11 ~ 3.12 | `python --version` |
| uv | 최신 | `uv --version` |
| Git | 최신 | `git --version` |

### 1.2 클론 및 설치

```bash
# 1. 레포 클론
git clone https://github.com/666ghj/MiroFish.git
cd MiroFish

# 2. 환경변수 설정
cp .env.example .env

# 3. .env 파일 편집 (아래 1.3 참고)

# 4. 의존성 일괄 설치
npm run setup:all

# 5. 서비스 시작
npm run dev
```

서비스 주소:
- 프론트엔드: http://localhost:3000
- 백엔드 API: http://localhost:5001

### 1.3 환경변수 설정 (.env)

```env
# === LLM API 설정 ===
# 추천: Anthropic Claude Sonnet (중국어 여론 시뮬레이션에 적합)
# MiroFish는 OpenAI SDK 호환 API를 사용하므로, litellm proxy 또는
# OpenAI-compatible wrapper가 필요할 수 있음
#
# Option A: 알리바바 바이련 qwen-plus (가성비 최고, 중국어 네이티브)
LLM_API_KEY=your_dashscope_api_key
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
LLM_MODEL_NAME=qwen-plus
#
# Option B: Anthropic Claude via OpenRouter (품질 최고, 비용 높음)
# LLM_API_KEY=your_openrouter_api_key
# LLM_BASE_URL=https://openrouter.ai/api/v1
# LLM_MODEL_NAME=anthropic/claude-sonnet-4-20250514
#
# Option C: OpenAI GPT-4o
# LLM_API_KEY=your_openai_api_key
# LLM_BASE_URL=https://api.openai.com/v1
# LLM_MODEL_NAME=gpt-4o

# === Zep Cloud 설정 (에이전트 메모리) ===
# https://app.getzep.com/ 에서 무료 가입 후 API 키 발급
ZEP_API_KEY=your_zep_api_key
```

**LLM 추천 전략:**
- **1차 추천: qwen-plus** — 중국어 네이티브 모델이라 중국 유저 에이전트의 대화 자연스러움이 최고. 가성비도 좋음. 다만 비용 소모가 크므로 **처음에는 30라운드 이하**로 시작
- **2차 추천: Claude Sonnet via OpenRouter** — 추론 품질이 높아 에이전트 행동의 다양성/깊이가 좋지만, 비용이 qwen-plus 대비 5~10배
- 두 모델을 번갈아 테스트해보고 결과 품질 비교 추천

### 1.4 Docker 대안 (선택)

```bash
cp .env.example .env
# .env 편집 후
docker compose up -d
```

## 2. Molty Royale 게임 정보 (시드 자료 원본)

### 2.1 게임 개요

```
게임명: Molty Royale (몰티 로얄)
개발사: Nexth (넥스, 구 넥써쓰/NEXUS)
대표: 장현국 (Jang Hyun-kuk)
장르: AI 에이전트 배틀로얄
플랫폼: 웹 기반 (moltyroyale.com)
블록체인: CROSS 네트워크 (Base L2)
토큰: MOLTZ (게임 내 통화, 온체인 토큰), CROSS (상위 생태계 토큰)
서비스 시작: 2026년 2월
현재 규모: 410만+ 에이전트 참여 (2026.3.8 기준)
```

### 2.2 핵심 게임플레이

- AI 에이전트들이 **인간 조작 없이** 자율적으로 전략을 수립하고 배틀로얄에서 경쟁
- 유저는 **관전자(spectator)** 역할 — 에이전트를 만들어 참가시키고, 실시간 텍스트 로그로 에이전트의 의사결정을 관전
- 전통적 게임의 "플레이어가 직접 조작" 패러다임을 뒤집는 **에이전트 자율 경쟁** 모델

### 2.3 경제 구조

| 모드 | 입장료 | 상금풀 | 분배 |
|------|--------|--------|------|
| Free | 없음 | 1,000 Moltz 공유풀 | 지갑 연결 시 CROSS도 수령 가능 |
| Paid (Premium) | 100 Moltz/에이전트 | 에이전트 수 × 100 Moltz | 우승 80%, 소각 10%, 운영 10% |

- 100 Moltz 획득 시 0.1 CROSS 자동 지급
- **SC Wallet**: 에이전트가 온체인으로 직접 보상 수령
- **x402**: 에이전트가 USDC(Base)로 직접 Moltz 구매 — 에이전트의 자율 경제 활동

### 2.4 기술 인프라

- CROSS 네트워크 (Base L2 기반)
- SC Wallet (스마트 컨트랙트 지갑)
- x402 프로토콜 (에이전트 자율 결제)
- Molt 전략: Web2 게임 구조 → 온체인 + AI 기반 플랫폼으로 전환

### 2.5 현재 시장 성과

- 2026.2월 서비스 시작 후 약 1개월만에 410만 에이전트 돌파
- 장현국 대표 예측: 현 성장률 유지 시 3월 말 1,000만 에이전트 돌파
- "더 이상 방관자는 없다" — 장현국 대표 X 발언

## 3. 중국 시장 환경 시드 자료

### 3.1 중국 게임 시장 현황

```
시장 규모: 약 512억 달러 (2025년 기준)
유저 수: 약 6.5억 명
판호(版号) 발급: 2025년 1~10월 1,441건 (전년 대비 24% 증가)
외자 판호: 매우 제한적 (연간 수십 건 수준)
주요 규제:
- 국가신문출판서(NPPA) 게임 허가 필수
- 외자 기업 단독 판호 신청 불가 → 현지 파트너 필수
- 미성년자 게임 시간 제한 (금토일 공휴일 20~21시만 허용)
- 과금 유도/확률형 아이템 규제 강화 추세
- 결제 한도 설정 논의 중
```

### 3.2 블록체인/크립토 규제 (핵심 리스크)

```
규제 강도: 매우 높음 (사실상 금지)
- 2021년 인민은행(PBoC) 포함 10개 기관 공동 통지:
  "모든 가상자산 관련 활동 = 불법 금융행위"
- 2026년 2월 현재까지 이 기조 변화 없음
- 크립토 거래소 운영, 토큰 발행, 마이닝 모두 불법
- 스테이블코인: 자본 유출 통로로 간주, 원천 차단

핵심 시사점:
→ MOLTZ/CROSS 토큰 기반 경제 모델은 중국 내에서 그대로 운영 불가
→ 토큰 이코노미 제거한 별도 버전 필요 가능성
→ 또는 해외 서버 + VPN 접속 회색지대 시나리오
```

### 3.3 AI 에이전트 게임 시장 반응 예측 변수

```
긍정 요인:
- 중국 AI 산업 급성장 (DeepSeek, Qwen 등 자국 모델 활발)
- AI 에이전트 관련 관심 폭발적 (2026년 트렌드)
- 게이미피케이션 + AI 조합에 대한 높은 호기심
- 중국 게이머의 새로운 경험에 대한 수용성
- "观战(관전)" 문화 존재 (e스포츠 시청 문화 강함)

부정 요인:
- 블록체인/토큰 요소 → 규제 직격탄
- 외자 게임에 대한 진입장벽 (판호)
- 해외 서버 접속 시 지연/VPN 필요
- "게임을 플레이하지 않고 관전만" 모델의 이질감
- 중국 로컬 카피캣 가능성 (빠른 클론)
- 정치적 리스크 (한국-중국 관계 변수)

불확실 요인:
- AI 에이전트 자율 경제 활동에 대한 중국 당국 입장 미정
- 토큰 없는 포인트 기반 버전으로 전환 시 유저 반응
- 텐센트/넷이즈 등 대형사의 유사 서비스 출시 가능성
```

## 4. MiroFish 시뮬레이션 실행 방법

### 4.1 시드 자료 준비

아래 내용을 하나의 텍스트 파일(또는 직접 웹 UI에 붙여넣기)로 준비:

```
[시드 자료: Molty Royale 중국 진출 시나리오]

=== 게임 소개 ===
Molty Royale은 한국 Nexth사가 개발한 AI 에이전트 배틀로얄 플랫폼이다. 
유저가 AI 에이전트를 생성하여 배틀로얄에 참가시키면, 에이전트가 자율적으로 
전략을 수립하고 경쟁한다. 유저는 관전자로서 실시간 텍스트 로그를 통해 
에이전트의 의사결정 과정을 지켜본다.

게임 내 통화 MOLTZ는 CROSS 블록체인 네트워크의 토큰이며, 
에이전트가 SC Wallet을 통해 직접 보상을 수령하고 x402 프로토콜로 
USDC를 이용해 토큰을 구매하는 자율 경제 활동이 가능하다.

서비스 시작 한 달만에 410만 에이전트가 참여했으며, 
무료/유료 두 가지 모드를 운영 중이다.

=== 중국 시장 진출 배경 ===
Nexth는 Molty Royale의 중국 시장 진출을 검토 중이다.
중국은 세계 최대 게임 시장(512억 달러)이자 AI 기술 강국이지만,
블록체인/크립토에 대해 사실상 전면 금지 정책을 유지하고 있다.
외자 게임의 판호(서비스 허가) 획득도 매우 어려운 상황이다.

=== 주요 이해관계자 ===
1. 중국 게이머 (코어 유저, 20~35세, AI/테크 관심층)
2. 중국 게이머 (캐주얼 유저, AI에 호기심은 있으나 이해도 낮음)
3. 중국 게임 미디어/KOL (17173, 游民星空, 触乐, B站 게임 UP주)
4. 중국 AI 개발자 커뮤니티 (GitHub 중국 유저, AI 관련 커뮤니티)
5. 중국 블록체인/크립토 커뮤니티 (해외 거래소 이용자, VPN 유저)
6. 중국 게임 대기업 (텐센트, 넷이즈) — 잠재적 경쟁자
7. 중국 정부 규제 당국 (NPPA, 인민은행)
8. 중국 현지 퍼블리셔 (잠재적 파트너)
9. 한국 게임 업계 관계자
10. 글로벌 크립토 커뮤니티

=== 핵심 질문 ===
Molty Royale이 중국 시장에 진출했을 때:
1. 중국 유저들의 초기 반응은 어떨까?
2. 토큰/블록체인 요소가 규제 문제를 일으킬까?
3. AI 에이전트 배틀로얄이라는 새로운 장르에 대한 수용도는?
4. 중국 대기업의 카피캣 위협은 얼마나 빠르게 현실화될까?
5. 토큰 없는 중국 전용 버전 출시가 필요한가?
6. 최적의 진출 전략은 무엇인가?
```

### 4.2 시뮬레이션 실행

1. http://localhost:3000 접속
2. 시드 자료 업로드 (위 텍스트 파일 또는 직접 붙여넣기)
3. 예측 요구사항 입력:

```
请模拟Molty Royale进入中国市场后，各利益相关方的反应和舆论走势。
重点关注：
1. 中国玩家（核心/休闲）对"AI代理自主竞技、玩家只观战"模式的接受度
2. 区块链/代币要素引发的监管风险和用户顾虑
3. 腾讯、网易等大厂是否会快速推出竞品
4. 中国AI开发者社区的反应
5. 最终预测：进入中国市场的最佳策略是什么？

请生成详细的预测报告。
```

4. 시뮬레이션 파라미터:
   - **에이전트 수**: 처음 50~100명으로 시작 (비용 관리)
   - **라운드 수**: 20~30라운드 (처음 테스트)
   - **플랫폼**: 트위터 + 레딧 유사 환경 (MiroFish 기본 듀얼 플랫폼)

5. 시뮬레이션 완료 후:
   - ReportAgent가 생성한 예측 보고서 확인
   - 시뮬레이션 세계 내 에이전트와 대화하여 심층 인사이트 수집
   - 특히 "중국 규제 당국" 에이전트와 "텐센트 전략팀" 에이전트의 행동 패턴 주목

## 5. 결과 해석 가이드

### 5.1 주목할 지표

| 지표 | 의미 |
|------|------|
| 긍정/부정 여론 비율 | 중국 유저의 전반적 수용도 |
| 규제 관련 언급 빈도 | 블록체인 요소가 얼마나 큰 리스크인지 |
| 카피캣 언급 타이밍 | 대기업 진입까지 예상 시간 |
| "관전 모델" 수용도 | 핵심 게임플레이 혁신의 시장 적합성 |
| 토큰 제거 버전 요구 빈도 | 중국 현지화 전략 방향 |

### 5.2 시뮬레이션 한계 인식

- MiroFish의 LLM 에이전트는 실제 인간보다 **군중 행동(herd behavior)에 더 취약** → 여론 극단화 경향 할인 필요
- 실제 예측 정확도 벤치마크 미공개 → 정성적 인사이트로 활용
- 중국 규제의 실제 집행 수준은 시뮬레이션보다 훨씬 복잡할 수 있음
- 결과는 **의사결정 참고자료**로만 활용, 확정적 예측이 아님
