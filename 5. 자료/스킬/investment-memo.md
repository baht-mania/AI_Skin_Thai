# investment-memo

넥써쓰 투자 검토 메모 프레임워크. Verse8/V8, Raid Labs 등 실제 협상 패턴 반영.

## 투자 유형
전략적 / 재무적 / 토큰 투자 / SAFE·컨버터블 노트 / 공동투자(Co-invest)

## 표준 메모 구조
1. 딜 개요 — 금액, 밸류에이션, 라운드, 공동투자자
2. 회사 현황 — 팀, 제품, 트랙션, 기존 투자자
3. 밸류에이션 분석 — 비교 Comps + DCF 요약
4. 투자 조건 — 지분율, 우선주, 이사회, 반희석, 락업
5. 협상 포지션 — 희망 조건 / 상대 제안 / BATNA / 레버리지
6. 전략적 시너지 — 퍼블리싱 연계, WEMIX/$CROSS, 네트워크
7. 리스크 매트릭스
8. 종합 의견 — 추진/조건부/보류/거절 + 대표 결정 요청

## 넥써쓰 내부 기준
- 목표 배수: 3x~5x
- 기대 회수 기간: 3~5년
- 최소 IRR: 20%+

## SAFE 체크리스트
- [ ] 밸류에이션 캡 설정
- [ ] 디스카운트율 15~20%
- [ ] MFN 조항
- [ ] 전환 트리거 명확화
- [ ] Pro-rata 권리

## 토큰 투자 주의사항
> ⚠️ 토큰만으로 대가를 받는 구조는 대표 방침상 금지. 반드시 현금 포함 구조로 협상.
- 투자자 락업 최소 12개월
- 전체 발행량 대비 배분 비율 확인
