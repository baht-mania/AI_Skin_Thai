# ceo-report-template

장현국 대표 보고용 문서/이메일 고정 포맷.

## 보고 원칙
1. 결론 선행 — Executive Summary에 핵심·요청 먼저
2. 데이터 중심 — 수치·근거 없는 주장 금지
3. 간결성 — 문단보다 번호/불릿
4. 리스크 명시 — 부정적 정보 숨기지 말 것
5. 액션 명확화 — 대표가 해야 할 결정/승인 마지막에

## 표준 구조

```
[Executive Summary]
현황 / 권고 / 요청 — 3~5문장

1. {섹션} — 현황 / 분석 / 시사점
2. {섹션} — 동일 구조

[종합 의견]
Option A (권장) / Option B

[리스크]
항목 | 가능성 | 영향도 | 대응

[액션 요청]
1. {결정 사항} — 기한
2. {승인 사항}
```

## 유형별 변형
- **주간 보고**: Slack #_weekly_business — 완료/진행중/계획/이슈
- **투자·BD**: 투자 조건 요약 + 협상 포지션 추가
- **퍼블리싱 검토**: KPI + 계약 조건 + 전략 적합성 추가
- **긴급 이슈**: 이슈 요약 → 현황 → 즉각 결정 (3단 단순화)

## 금지 표현
- ❌ "~인 것 같습니다" → ✅ "~입니다"
- ❌ 배경 설명으로 시작 → ✅ 결론으로 시작
- ❌ "조만간", "빠른 시일 내" → ✅ 구체적 날짜
- ❌ 리스크 누락/축소 → ✅ 투명하게 명시
